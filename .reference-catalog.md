# Pim Pam Pum — Content Reference Catalog

Compact catalog of all characters, cards, effects and equipment from the class-based system, for re-authoring as skills/actions.

Notes on notation:
- `phys`/`mag`/`def` show the DiceRoll as `XdY` or `XdY+Z`/`XdY-Z` (e.g. `new DiceRoll(1,4,-1)` => `1d4-1`; `new DiceRoll(0,0,1)` => `+1` flat; `new DiceRoll(2,4,-2)` => `2d4-2`).
- `effect` lists the SpecialEffect `type` plus key params; CharacteristicModifier abbreviated as `CharMod`.
- Stat icons abbreviated F/M/D/V; PV = lives.
- Card icon paths share prefix `icons/000000/transparent/1x1/`; only the `[artist]/[name].svg` tail is shown per card.

## EFFECT TYPES USED

Distinct SpecialEffect `type` strings that actually appear on cards/equipment in the above content (the union in card.ts also declares unused variants — those are listed separately at the end).

- **None** — no special effect (plain attack/defense). Used by: Fighter (Tall d'espasa, Posició defensiva), Wizard (Bola de foc), Rogue (Ballesta, Foc alquímic), Druid (Fuet d'espines, Raig de lluna, Urpada salvatge, Pell d'escorça), Goblin (Punyalada ràpida, Protegir el clan), Goblin Shaman (Llamp), Basilisk (Escames impenetrables), Spined Devil (Cortina de foc), Bone Devil (Esgarrapada, Defensa esquelètica), Stone Golem (Cop de pedra), Wolf (Urpa ràpida, Protegir la manada).
- **CharacteristicModifier** (CharMod) — unified stat buff/debuff: array of {characteristic strength|magic|defense|speed, amount, optional dice} applied to target self|allies|team|enemy|enemies for a ModifierDuration. Used by: Fighter (Crit de guerra), Wizard (Polimorfisme), Rogue (Bomba de fum), Barbarian (Ràbia), Cleric (Benedicció), Monk (Concentració de ki), Bard (Harmonia protectora, Balada heroica), Warlock (Raig corrosiu), Paladin (Escomesa sagrada, Jurament sagrat), Druid (Lligams de natura), Sorcerer (Distorsió temporal), Goblin Shaman (Possessió demoníaca), Horned Devil (Flames de l'avern), Stone Golem (Enduriment), Poció de velocitat.
- **MultiTarget** {count} — attack hits up to N chosen enemies. Used by: Fighter (Atac ampli), Wizard (Cadena de llamps), Rogue (Dagues), Barbarian (Gran escomesa), Goblin Shaman (Pluja de flames), Basilisk (Cop de cua), Horned Devil (Alè de l'infern), Stone Golem (Terratrèmol).
- **DefendMultiple** {count} — defense card protects N chosen allies at once. Used by: Wizard (Pantalla protectora, count 2), Paladin (Aura protectora, count 10 = whole team).
- **ActionSurge** {secondAttackDice} — after attacking, immediately makes a second attack vs a random enemy. Used by: Fighter (Escomesa furiosa).
- **SecondWind** {healAmount, defenseBoost} — heal lives + temporary defense boost this turn. Used by: Fighter (Segon alè).
- **PiercingStrike** — attack ignores Defense cards; always hits the original target. Used by: Wizard (Míssils arcans), Monk (Puntada voladora).
- **Counterspell** — choose an enemy; if they play a magic or Focus card it is cancelled (no effect on physical/defense). Used by: Wizard (Contrahechís).
- **CoordinatedAmbush** — choose an enemy; all allies attacking it this turn get +1d8+2 to their roll. Used by: Rogue (Emboscada coordinada).
- **CounterThrow** — on defending, attacker gets V-3 next turn. Used by: Rogue (Clon de fum).
- **DodgeWithSpeedBoost** — dodge all attacks this turn; V+5 and F+4 next turn. Used by: Rogue (El·lusió).
- **PoisonWeapon** — choose 3 allies (incl. self); their physical attacks cost an extra life for rest of combat. Used by: Rogue (Enverinar arma).
- **RecklessAttack** {defenseReduction} — strong attack; lose defense this and next turn. Used by: Barbarian (Atac Temerari).
- **Frenzy** {bonusDicePerLostLife} — attack gains bonus dice per life already lost. Used by: Barbarian (Frenesia).
- **BerserkerEndurance** {strengthBoost, speedBoost?} — defense card; if it takes damage, gain F+/V+ for rest of combat. Used by: Barbarian (Venjança).
- **IntimidatingRoar** {dice, threshold} — per opponent: if F+dice > enemy F+M+threshold, enemy is stunned (action cancelled) this turn. Used by: Barbarian (Rugit intimidant).
- **TouchOfDeath** {strengthDebuff, magicDebuff} — attacked target gets F-/M- next turn. Used by: Cleric (Toc de la mort).
- **LifeDrain** — if attack hits, attacker heals 1 life. Used by: Cleric (Drenatge vital), Warlock (Raig vampíric).
- **DeathCurse** {dice} — choose an enemy; self+allies get +dice attacking it for rest of combat. Used by: Cleric (Maledicció mortal).
- **ShroudDebuff** {amount} — on defending, attacker gets F-amount next turn. Used by: Cleric (Mantell diví).
